# Roadmap

[`pipeline_view.html`](pipeline_view.html) is an interactive, collapsible view of
the full development pipeline — every system grouped by area (Combat, Macro, Map,
Town, Overworld, Unlocks, …), color-coded by status, with priority weights and
design notes on each item. Open it in a browser; "Toggle All Notes" expands the
reasoning behind each task.

It doubles as a design journal: the notes capture not just *what* is planned but
*why*, including ideas that were considered and cut (e.g. real-time trainer
control, a tower-defense minigame) and the open questions still being weighed.

*Note: the pipeline tracks intent at an abstract level. Specific goals and scope
inside it evolve as the project does — treat it as a living snapshot of
direction, not a fixed contract.*
