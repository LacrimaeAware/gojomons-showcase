# Media

Local source files for the clips and stills shown in the main README.

- Video (`.mp4`) is not committed. It's uploaded to GitHub's attachment CDN and
  referenced by URL in the README, so the files stay out of the repo.
- Published stills (currently `StyleExploration.png`) are committed and embedded
  by path.

The `.gitignore` here keeps everything out of git except the committed stills and
this file.
